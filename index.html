Git-Hub Commands

Git Push

Git push wird dazu verwendet um lokale Veränderungen in ein zentrales Repository hochzuladen und zu veröffentlichen. Weitere Teammitglieder können dann die Veränderung herunterladen

Git Branch

Verweist auf einen bestimmten Commit kann durch „git branch“ eröffnet werden, durch „git checkout“ können diese mit dem Head „ausgewählt“ werden.
